## E2 civ ship character differentiation (plan only — no data changed)

Goal: give each civ ship a distinct character (e.g. Western = fast/raider, Chinese = slow/tank) while keeping them balanced with each other and with War Galley. All values below are **proposals**; no files are modified until you approve.

### Balance principle

- Keep the same **effective power budget** across civs: trade HP vs DPS vs speed so that in aggregate they stay between WG E1 and E3 and comparable to each other.
- **Baseline (current E2):** HP 1370, Damage 41, Range 8, Reload 2.6, Build 72, Cost 150 F / 575 W / 400 G / 200 Tin. Ship speed in DDF is 1.4 (shared).

### Where stats live

| What | Where |
|------|--------|
| HP, LOS, Damage, Range, Reload, Build time, Cost | `EE2X_db\TechTree\upgrade_unittypes.csv` (per E2 upgrade row) |
| Ship speed (NavalMove) | `EE2X_db\Units\Tao_Ship_FarEast.ddf` and `Tao_Ship_Western.ddf` — base `abilities` of each UnitType (MaChuan, JapanBoat2, Guichuan, WesternBoat2, AfricanBoat, MesoAmerBoat, MiddleEastBoat). Changing speed here affects that ship at every epoch where it is used. |

### Proposed character archetypes

1. **Western (WesternBoat2)** — Viking/raider  
   - Faster, lower HP, faster reload.  
   - Speed ↑, HP ↓, Reload ↓; Damage can stay or dip slightly so DPS stays balanced.

2. **Chinese (MaChuan)** — Large junk / flagship  
   - Slower, higher HP, slower reload.  
   - Speed ↓, HP ↑, Reload ↑; Damage can stay or rise slightly.

3. **Far East (JapanBoat2)** — Agile, balanced  
   - Slightly faster than baseline, moderate HP, slightly faster reload.  
   - Between Western and baseline.

4. **Korean (Guichuan)** — Turtle ship / sturdy  
   - Slightly slower, higher HP, same or slightly slower reload.  
   - Between Chinese and baseline.

5. **African (AfricanBoat)** — Raider / coastal  
   - Faster, lower HP, faster reload (similar idea to Western, different flavour).  
   - Can mirror Western’s stat style with small differences.

6. **MesoAmerican (MesoAmerBoat)** — Light / canoe heritage  
   - Fast, low HP, fast reload.  
   - Another “light raider” variant.

7. **Middle East (MiddleEastBoat)** — Trade / endurance  
   - Slower, higher HP, slower reload.  
   - Tanky, similar direction to Chinese but not identical.

### Example stat bands (illustrative — to be tuned)

- **Baseline reference:** HP 1370, Dmg 41, Reload 2.6, Speed 1.4 → effective DPS ≈ 41/2.6 ≈ 15.8.  
- **“Raider” (e.g. Western):** HP ~1180, Dmg ~38, Reload ~2.1, Speed ~1.55. Same cost.  
- **“Tank” (e.g. Chinese):** HP ~1560, Dmg ~44, Reload ~3.0, Speed ~1.25. Same cost.  
- **“Middle” (e.g. Japanese):** HP ~1300, Dmg ~41, Reload ~2.4, Speed ~1.45.  

Keep **Range 8**, **LOS 9**, **Build 72**, and **Cost 150 F / 575 W / 400 G / 200 Tin** the same for all, so differentiation is only HP / Damage / Reload / Speed.

## E2 文明舰船性格分化方案（中文）

**目标：** 为各文明舰船赋予不同性格（如西方 = 快速/劫掠型，中国 = 慢速/肉盾型），同时保持彼此之间以及与战船（WG）的平衡。**已按上文 “Implemented values (final)” 表格实施。**

### 平衡原则

- 各文明保持相同的**有效战力预算**：在 HP、DPS、速度之间做取舍，使整体仍介于 WG E1 与 E3 之间且彼此可比。
- **基准（当前 E2）：** HP 1370，伤害 41，射程 8，装填 2.6，建造 72，成本 150 食 / 575 木 / 400 金 / 200 锡。DDF 中舰船速度为 1.4（共用）。

### 数值所在位置

| 项目 | 位置 |
|------|--------|
| HP、视野、伤害、射程、装填、建造时间、成本 | `EE2X_db\TechTree\upgrade_unittypes.csv`（每条 E2 升级行） |
| 舰船速度（NavalMove） | `EE2X_db\Units\Tao_Ship_FarEast.ddf` 与 `Tao_Ship_Western.ddf` 中各 UnitType 的基轴 `abilities`（马船、日本船2、龟船、西方船2、非洲船、美洲船、中东船）。在此修改速度会影响该舰在所有使用到的时代。 |

### 拟议性格类型

1. **西方 (WesternBoat2)** — 维京/劫掠型  
   - 更快、HP 更低、装填更快。  
   - 速度↑，HP↓，装填↓；伤害可不变或略降以保持 DPS 平衡。

2. **中国 (MaChuan)** — 大船/旗舰型  
   - 更慢、HP 更高、装填更慢。  
   - 速度↓，HP↑，装填↑；伤害可不变或略升。

3. **远东 (JapanBoat2)** — 灵活、均衡型  
   - 略快于基准，HP 适中，装填略快。  
   - 介于西方与基准之间。

4. **朝鲜 (Guichuan)** — 龟船/坚固型  
   - 略慢，HP 更高，装填不变或略慢。  
   - 介于中国与基准之间。

5. **非洲 (AfricanBoat)** — 劫掠/沿岸型  
   - 更快、HP 更低、装填更快（思路类似西方，风格不同）。  
   - 可与西方采用相近数值风格并做小幅差异。

6. **美洲 (MesoAmerBoat)** — 轻型/独木舟传统  
   - 快、HP 低、装填快。  
   - 另一种“轻劫掠”变体。

7. **中东 (MiddleEastBoat)** — 贸易/续航型  
   - 更慢、HP 更高、装填更慢。  
   - 偏肉盾，方向与中国类似但不完全相同。
