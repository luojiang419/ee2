## 迁移到其他设备时怎么用

1. 优先从本目录继续开发，不要再回到仓库根目录里混合修改魔改版与远航版文件。
2. 启动器源码开发：
   - 进入 `01-启动器源码/EE2X远航版启动器/`
   - 执行 `npm install`
   - 日常构建 `npm run build`
   - 正式安装包编译 `npm run build:release`
3. 游戏更新后端与安装包更新后端源码分别在 `03-更新后端/` 下。
4. 更新包打包/推送工具在 `04-更新包工具/` 下，远航版专用发布配置模板见：
   - `04-更新包工具/更新器套件/config/publish.yuanhang.example.json`
