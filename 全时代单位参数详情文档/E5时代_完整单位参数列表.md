## 七、跨文档修改速查表

### 修改任意E5单位时必查的4个位置

```
① CSV  → upgrade_unittypes.csv         "{UnitType}UpgradeEpoch5"行 → HP/DAMAGE/RANGE/COST
② DDF  → 见本索引各章节               UnitType块 → popCount/RPS/abilities覆盖
③ TECH → dbtechtreenode.csv             单位节点行 → 建造资源/时间/前置
④ EP5 → epoch5_upgrades.ddf  升级集引用 → 文明分配/自动研究
```

### DDF vs CSV 属性覆盖规则
- **popCount**: DDF覆盖CSV（船只人口在DDF中）
- **RPS类型**: DDF `rps` 字段决定
- **范围伤害/AOE**: DDF `AreaEffect`/`areaDamageRadius` 块
- **模型/动画**: 仅在DDF中
- **UpgradeAbilities**: DDF E{N} Attack升级会覆盖CSV的damage/range/reload
