# 根目录可执行文件 — 游戏主程序和DLL

## 基本信息
- **路径**: `Empire Earth II\` (根目录)
- **类型**: 游戏可执行文件和核心DLL
- **修改优先级**: 🔴 高（EXE包含硬编码数据） / ⚪ 不修改（DLL文件不可直接编辑）
- **修改方式**: EXE通过UP1.5补丁系统修改；DLL不可直接编辑

## 修改要点

### EE2X.exe 修改
- **不可直接编辑**二进制文件
- 通过 `Unofficial Patch Files\EXEGeneratorData\` 生成带补丁的EXE
- 使用 `提取EXE中ASCII字符串脚本.ps1` 搜索硬编码常量

### UP15.dll 不可修改
- 编译后的C++ DLL
- 控制UP1.5新单位的加载和游戏行为修改
- 如有源代码可重新编译

- 补丁系统: `Unofficial Patch Files\`
