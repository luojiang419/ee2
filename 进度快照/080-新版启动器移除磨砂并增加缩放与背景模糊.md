# 进度快照 080 — 新版启动器移除磨砂并增加缩放与背景模糊

> 日期: 2026-05-25

---

## 已完成内容

### 1. 已移除磨砂玻璃面板

- 修改文件：`EE2X魔改版启动器/src/styles.css`
- 修复点：
  - 移除 `backdrop-filter` / `-webkit-backdrop-filter`
  - `glass` / `glass-lite` / `modal` 等面板改为深色实体面板
  - 保留边框与阴影层级，但不再透过背景做磨砂模糊

### 2. 已增加背景模糊度控制

- 修改文件：
  - `EE2X魔改版启动器/src/App.tsx`
  - `EE2X魔改版启动器/src/lib/types.ts`
  - `EE2X魔改版启动器/src/lib/tauri.ts`
  - `EE2X魔改版启动器/src-tauri/src/lib.rs`
- 新增配置项：
  - `backgroundBlur`
- 设置页已增加滑杆：
  - 范围 `0 ~ 24px`
  - 实时显示当前模糊值
- 当前实现会把模糊同时应用到：
  - 首页背景图片 / 背景视频
  - 设置页背景预览

### 3. 默认窗口尺寸已改为 960×540

- 修改文件：`EE2X魔改版启动器/src-tauri/tauri.conf.json`
- 已调整：
  - `width = 960`
  - `height = 540`
  - `minWidth = 960`
  - `minHeight = 540`

### 4. 已增加整套 UI 自动缩放壳

- 修改文件：
  - `EE2X魔改版启动器/src/App.tsx`
  - `EE2X魔改版启动器/src/styles.css`
- 实现方式：
  - 新增 `app-viewport`
  - 以 `1600×960` 作为设计基准尺寸
  - 根据实际窗口尺寸自动计算 `uiScale`
  - 将整套启动器模块整体缩放后居中显示
- 结果：
  - 在 `960×540` 默认窗口下，模块会整体缩放，保持完整可操作
  - 不需要让用户手动拉大窗口才能看到主要操作区

### 5. 构建验证

- `npm run build:frontend`：通过
- `cargo check`：通过
- `npm exec tauri build`：通过
