## 已完成内容

### 1. 启动器安装包更新已改为强制升级

- 启动流程已调整为：
  - 先处理游戏版本同步
  - 再检查启动器安装包更新
- 若发现新的启动器安装包版本：
  - 自动后台下载
  - 下载完成后以阻塞式弹窗要求立即升级
  - 不再允许“稍后更新”后继续进入软件首页

### 2. 无实际游戏更新时不再弹“更新完成”

- 已修正启动时逻辑：
  - 不再把旧 launcher 标记误当成游戏更新
  - 仅当 `appliedVersions.length > 0` 时才显示“更新完成”弹窗
- 因此：
  - 游戏本身没更新时，不再每次启动都弹完成窗

### 3. 1.0.26 已正式发布到 3011

- 当前 latest: `1.0.26`

- 当前 `output` 目录已包含：

  - `EE2X魔改版启动器_1.0.26_x64-setup.nsis.zip`
  - `EE2X魔改版启动器_1.0.26_x64-setup.nsis.zip.sig`

### 4. 构建发布链已补充自动签名依赖

- `build-release.mjs` 已增强：
  - 自动安装 Windows Python 的 `py-minisign`

  - 自动同步产物到 `output`
