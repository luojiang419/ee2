# 进度快照 098 — 新版启动器首页在线玩家列表内部滚动修复

> 日期: 2026-05-26

---

## 已完成内容

### 1. 修复首页在线玩家列表超出边界时无限拉长的问题

- 原因是左侧 `player-board` 容器没有被固定在可用高度内
- 导致内部 `board-table` 即使设置了滚动，也会跟着父容器一起被拉长

### 2. 已将在线玩家侧栏改为固定高度内部滚动

- `player-board` 现在改为：
  - `min-height: 0`
  - `height: 100%`
  - `overflow: hidden`
- 这样在线玩家列表在人数较多时会在面板内部滚动，不再撑破首页布局

### 3. 已完成构建验证

- `npm run build` 通过
