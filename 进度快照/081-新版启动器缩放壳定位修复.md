# 进度快照 081 — 新版启动器缩放壳定位修复

> 日期: 2026-05-25

---

## 已完成内容

### 1. 修复默认 960×540 下界面整体偏到右下角的问题

- 修改文件：
  - `EE2X魔改版启动器/src/App.tsx`
  - `EE2X魔改版启动器/src/styles.css`
- 根因：
  - 之前直接对 `1600×960` 主壳做 `transform: scale(...)`
  - 但主壳本身仍按未缩放尺寸参与父容器居中布局
  - 导致在小窗口下，视觉上只显示了被裁切后的右下区域

### 2. 缩放壳结构已重构

- 新增一层 `app-stage`
  - 尺寸等于“缩放后”的实际占位尺寸
- `app-shell`
  - 保持设计尺寸 `1600×960`
  - 改为 `position: absolute`
  - 使用 `transform-origin: top left`
- 结果：
  - 父容器居中的是“缩放后尺寸”
  - 真正渲染内容按左上角缩放，不再错位到右下角

### 3. 构建验证

- `npm run build:frontend`：通过
- `npm exec tauri build`：通过
- 当前安装包：
  - `EE2X魔改版启动器/src-tauri/target/release/bundle/nsis/EE2X魔改版启动器_1.0.1_x64-setup.exe`
