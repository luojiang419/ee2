# 进度快照 088 — 新版启动器启动强制更新同步

> 日期: 2026-05-25

---

## 已完成内容

### 1. 启动时已改为强制检查更新

- 启动流程现在会在进入正常使用前先执行版本检查
- 仅当游戏目录有效时执行强制启动更新检查
- 如果当前已经是最新版本，才继续后续登录/正常使用流程

### 2. 检测到更新后改为阻断式强制同步

- 如果检测到启动器或游戏存在可用更新：
  - 立即弹出阻断式更新同步弹窗
  - 自动执行更新下载与同步
  - 用户不能跳过更新继续进入软件
- 弹窗内会直接显示更新日志流和进度

### 3. 更新失败时只能重试

- 启动更新检查失败时，会显示阻断式错误状态
- 更新同步失败时，也会停留在阻断式错误状态
- 当前只提供“重试同步”，不提供跳过继续按钮

### 4. 与现有首启引导兼容

- 无目录时：先走“选择魔改版游戏目录”引导
- 目录有效后：先检查并强制同步更新
- 没有更新或更新完成重启后：再进入“登录 / 注册”引导或主界面

### 5. 已完成构建与安装包验证

- `npm run build` 通过
- `npm exec tauri build` 通过
- 已重新生成 `1.0.1` NSIS 安装包
