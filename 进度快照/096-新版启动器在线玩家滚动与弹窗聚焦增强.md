# 进度快照 096 — 新版启动器在线玩家滚动与弹窗聚焦增强

> 日期: 2026-05-26

## 已完成内容

### 1. 首页在线玩家列表改为内部滚动

- 首页左侧 `在线玩家` 面板现在不再随着玩家数量无限拉长
- 列表区域改为内部滚动容器：
  - 超出可见高度时，使用滚轮在面板内部滚动
  - 不再顶破首页布局边界
- 表头已固定为吸顶，滚动时仍可看到列名

### 2. 弹窗遮罩进一步增强背景模糊与暗化

- `overlay` 现在使用更重的背景遮罩
- 同时提高背景模糊强度
- 作用：
  - 更新完成弹窗更突出
  - 登录/注册弹窗更突出
  - 首启引导弹窗也会同步受益

### 3. 弹窗本体背景进一步加深

- `modal` 背景透明度已进一步降低
- 外阴影已增强
- 现在弹窗文字不再容易和背景图、背景日志混在一起

### 4. 已确认首页“一键更新”当前行为

- 当前首页右侧 `一键更新` 按钮已经是：
  - 点击后切到更新中心
  - 同时直接执行强制更新
- 本轮未再次改动该逻辑，已沿用现有实现

### 5. 已完成构建验证

- `npm run build` 通过
