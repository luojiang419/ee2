# 进度快照 102 — 新版启动器保存分辨率前端立即应用兜底

> 日期: 2026-05-26

---

## 已完成内容

### 1. 新增前端窗口尺寸立即应用兜底

- 在保存设置成功后
- 前端会直接读取当前选择的 `preferredResolution`
- 然后调用当前窗口 API：
  - `setSize(...)`
  - `center()`
- 确保用户点击“保存设置”后，窗口尺寸立刻切到所选分辨率

### 2. 补齐窗口尺寸调整所需权限

- capability 已新增：
  - `core:window:allow-set-size`
  - `core:window:allow-center`
- 避免前端直接调用窗口尺寸调整 API 时被权限拦住

### 3. 与现有 Rust 侧分辨率恢复逻辑并存

- Rust 侧仍然保留：
  - 启动时按 `preferredResolution` 恢复窗口尺寸
  - 保存配置时尝试同步应用窗口尺寸
- 本次新增的是前端“立即生效”兜底层
- 这样能覆盖用户当前保存时立即看见窗口变化的需求

### 4. 已完成构建验证

- `npm run build` 通过
- Rust `release` 构建通过
