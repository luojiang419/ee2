## 已完成内容

### 1. 页面访问即弹出账号登录遮罩

- Web 推送页现在在页面加载/刷新时会直接显示居中登录遮罩
- 未登录前无法进入操作界面
- 当前实现为：
  - 页面初始化先清掉当前标签页旧登录态
  - 再弹出登录层
- 默认账号密码仍为：
  - `ee2x / ee2x`

### 2. 删除确认已从浏览器原生框改为页面中心弹窗

- 不再使用浏览器原生 `confirm`
- 删除版本时会显示页面居中的自定义确认弹窗
- 当前版本删除提示文案仍为：
  - 删除后会自动回退到上一版

### 3. 当前版本与历史栏已改为固定高度滚动区

- 历史列表外层已加固定高度容器
- 超出部分改为内部滚动，不再无限拉长整页
- 当前样式目标是约 5 条历史记录的可视高度
- 历史说明文本已压缩为两行截断，保证列表更稳定

### 4. 顶部已改为当前账号状态 + 退出登录

- 页头右侧现在显示：
  - 固定频道
  - 当前账号
  - 退出登录按钮
- 退出登录后会重新弹出登录遮罩
