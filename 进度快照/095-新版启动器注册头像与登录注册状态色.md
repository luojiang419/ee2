# 进度快照 095 — 新版启动器注册头像与登录注册状态色

> 日期: 2026-05-26

---

## 已完成内容

### 1. 登录 / 注册切换按钮已改为明确状态色

- 默认进入认证弹窗时，固定为“登录”态
- 登录按钮：
  - 选中时深蓝色
  - 未选中时深灰色
- 注册按钮：
  - 选中时深蓝色
  - 未选中时深灰色
- 首启向导认证卡片和普通认证弹窗已统一使用同一套颜色规则

### 2. 登录态与注册态已拆成明确双态表单

- 登录态只显示：
  - 用户名
  - 密码
  - 底部按钮：`登录账号`
- 注册态显示：
  - 头像上传与预览
  - 用户名
  - 密码
  - 底部按钮：`注册并登录`

### 3. 已接入注册头像上传预览

- 注册态新增隐藏文件选择框
- 选择头像后会立即显示本地圆形预览
- 提交注册前会把头像文件转换成 data URL/base64 字符串
- 若用户不选头像，则继续提交空字符串，保持旧接口兼容

### 4. 已按旧版启动器注册接口兼容实现

- 已确认旧版启动器注册时会向 `/api/register` 提交：
  - `username`
  - `password`
  - `avatar`
  - `computer_name`
  - `ip`
- 新版本轮已补齐 `avatar` 字段
- Rust 侧 `auth_register` 现在接收：
  - `username`
  - `password`
  - `avatar`
- 注册成功后：
  - 优先使用服务端返回的头像
  - 若服务端未回显头像，则回退使用本地提交值写入 session

### 5. 已完成构建与安装包验证

- `npm run build` 通过
- `npm run build:release` 通过
- 版本已自动递增到 `1.0.6`
- 已生成安装包：
  - `EE2X魔改版启动器_1.0.6_x64-setup.exe`
