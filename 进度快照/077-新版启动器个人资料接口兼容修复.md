# 进度快照 077 — 新版启动器个人资料接口兼容修复

> 日期: 2026-05-25

---

## 已完成内容

### 1. 个人资料接口解析错误已修复

- 修改文件：`EE2X魔改版启动器/src-tauri/src/lib.rs`
- 修复点：
  - 后端 `profile` 接口返回的 `is_online` 实际是 `1/0`
  - 新版启动器原先按 `bool` 严格反序列化，导致整包资料解析失败
  - 现在已改为兼容：
    - `true / false`
    - `1 / 0`
    - `"true" / "false" / "1" / "0"`

### 2. 影响结果

- `profile` 请求成功时，不再因为 `is_online: 1` 报：
  - `invalid type: integer '1', expected a boolean`
- 因为 `profile` 现在能正常解析：
  - 等级
  - 战力
  - 胜场
  - 注册时间
  - 上次登录
  - 总在线时长
  将能正常显示
- 同时也不会再因为 `profile` 失败而继续把 `runtime summary 404` 噪声暴露在个人资料弹窗里

### 3. 构建验证

- `cargo check`：通过
- `npm exec tauri build`：通过
- 产物：
  - `EE2X魔改版启动器/src-tauri/target/release/ee2x_mod_launcher.exe`
  - `EE2X魔改版启动器/src-tauri/target/release/bundle/nsis/EE2X魔改版启动器_0.1.0_x64-setup.exe`
