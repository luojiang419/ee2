## 已完成内容

### 1. 文件勾选区已从平铺视图改成树形目录结构

- 现在按原游戏目录层级显示
- 不再把全部文件平铺在一层
- 根目录本身不显示为可勾选项，只显示其下真实目录与文件

### 2. 树形交互已落地

- 文件夹支持展开 / 折叠
- 新增工具按钮：
  - `全部展开`
  - `全部折叠`
  - `清空勾选`
- 勾选文件夹时采用递归全选：
  - 自动选中全部子文件
  - 自动选中全部子目录
- 部分子项被选中时，父目录显示半选中状态

### 3. 搜索仍保留层级

- 搜索命中深层文件时，会保留祖先目录可见
- 搜索结果不再退化成拍平列表

### 4. 发布输出仍保持扁平文件清单

- 树形结构只用于 UI
- 真正写入 `selection.txt` 的仍然是叶子文件路径
- 不会把目录节点写入发布清单

### 5. 统计卡片已按最终文件选择数工作

- `启动器文件数`
- `游戏文件数`

现在统计基于最终选中的文件，而不是目录树节点数

### 6. 旧的列表/缩略切换已移除

- 不再保留 `列表 / 缩略` 分段按钮
- 文件勾选区统一为树形目录浏览模式

- Windows Flutter 构建已成功
