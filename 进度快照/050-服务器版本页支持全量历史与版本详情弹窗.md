# 进度快照 050 — 服务器版本页支持全量历史与版本详情弹窗

> 日期: 2026-05-21

---

## 已完成内容

### 1. 服务器版本标签页已升级为全量历史模式

- `更新器/flutter_publish_tool/lib/main.dart`
- 原先只拉取最近 20 条服务器历史
- 现在改为拉取当前频道全部已发布版本
- 页面顶部新增更明确的历史数与累计下载量展示
- 版本卡片可直接用于核对版本号是否冲突，也可用于查阅历史更新记录

### 2. 版本卡片已支持详情弹窗

- 点击 `服务器版本` 页里的任意版本卡片
- 会弹出版本详情窗口，显示：
  - 版本号
  - Release ID
  - 推送时间
  - 最后下载时间
  - 启动器/游戏包大小
  - 启动器/游戏包下载次数
  - 版本下载量
  - 文件变更与删除变更
  - 完整更新内容

### 3. 更新后端已补齐版本历史详情字段

- `更新器/update_backend_mg/app/db.py`
  - 新增 `release_package_downloads` 表
- `更新器/update_backend_mg/app/service.py`
  - `history` 支持 `limit=0` 返回全部历史
  - 历史项新增：
    - `publishedAt`
    - `required`
    - `launcherPackageSize`
    - `gamePackageSize`
    - `launcherDownloadCount`
    - `gameDownloadCount`
    - `downloadCount`
    - `totalDownloadCount`
    - `lastDownloadedAt`
- `更新器/update_backend_mg/app/main.py`
  - `/updates/...` 下载入口改为显式 FastAPI 路由
  - 下载 ZIP 时自动累计对应版本包的下载次数

### 4. 其他入口已同步适配

- `更新器/update_backend_mg/app/publish/assets/app.js`
  - Web 推送页历史读取改为全量
  - 历史项增加下载次数展示
- `更新器/src/ee2x_update_suite/publisher/http_backend.py`
- `更新器/src/ee2x_update_suite/bridge/__main__.py`
  - 历史接口默认值改为 `limit=0`
- 文档已同步更新：
  - `更新器/update_backend_mg/README.md`
  - `更新器/docs/发布协议.md`
  - `更新器/docs/服务器部署说明.md`
