## 已完成内容

### E14海军RPS四角克制链

CSV新增51行RPS规则，建立海军剪刀石头布克制体系：

```
潜艇(140%) → 主力舰(125%) → 驱逐舰(130%) → 潜艇
潜艇(130%) → 航母(125%) → 主力舰
驱逐舰(80%) → 航母（劣势）
```

### 系数详情

| 攻→守 | 系数 | 覆盖时代 |
|:------|:----:|:--------|
| 潜艇 → 主力舰 | **140%** | E13/E14/E15 |
| 俄罗斯akula → 主力舰 | **150%** | E14（特色超额） |
| 潜艇 → 航母 | **130%** | E13/E14/E15 |
| 主力舰 → 驱逐舰 | **125%** | E13/E14/E15 |
| 驱逐舰 → 潜艇 | **130%** | E13/E14/E15（翻转原85%反克制） |
| 驱逐舰 → 航母 | **80%** | E13/E14/E15（劣势） |
| 航母 → 主力舰 | **125%** | E13/E14/E15 |

### 覆盖单位

- 通用: Submarine (E13/E15), Destroyer (E13/E14/E15), Battleship (E13/E14/E15), AircraftCarrier (E14)
- 中国: Type039B (E14), Ch055/Ch055A (Battleship), Ch052d (Destroyer), ChinaCarrier
- 美国: VirginiaSubmarine (E14), Ticonderoga/DDG1001 (Battleship), DDG123 (Destroyer), AircraftCarrier
- 俄罗斯: akula (E14, 150%), kirov/Cruiser1164 (Battleship), DestroyerLeader (Destroyer), RussianCarrier

### 三国特色

- **中国**: 通用系数，055装填最快(1.9)实现DPS优势
- **美国**: 通用系数，Ticonderoga面板最强(HP18400/伤害1260/射程26)自然压制
- **俄罗斯**: akula对主力舰150%（超额），补偿kirov面板最弱但水下优势
