## 已完成内容

### 1. 启动器配置已拆成“公共默认配置 + 本地私有配置”

- 新增公共配置文件：
  - `地球帝国二代远航版启动器/Defaults/launcher-public.json`
- 启动器 `main.js` 现在按三层读取配置：
  1. 代码内置默认值
  2. `Defaults/launcher-public.json`
  3. `Config/ee2x-launcher.json` / `Config/ee2x-settings.json`
- 公共托管键现在只来自 `Defaults/launcher-public.json`：
  - `serverUrl`
  - `userServerUrl`
  - `updateServerHttp`
  - `serverUrlPrimary`
  - `serverUrlBackup`
  - `updateServerWs`
  - `networkServer`
  - `battleApiUrl`
  - `battleApiModel`
- `saveConfig()` 已改成只写本地私有键，不再把公共托管键写回 `Config/*.json`

### 2. 启动器本地身份与状态文件继续受保护

以下内容继续禁止推送：

- `地球帝国二代远航版启动器/Config/**`
- `地球帝国二代远航版启动器/Logs/**`
- `地球帝国二代远航版启动器/data/userdata/**`
- `地球帝国二代远航版启动器/data/game-csv/**`
- `地球帝国二代远航版启动器/data/Settlement-img/**`
- `地球帝国二代远航版启动器/update/runtime/**`

这保证：

- `ee2x-user.json` 登录 token 不会随更新包下发
- `ee2x-device-id` / `ee2x-device-meta.json` 不会被复制给其他玩家
- 本机 `gameExePath` / `gameDir` 绝对路径不会污染别人机器

### 3. 发布核心规则已支持“自动排除”与“显式硬拦截”

- `更新器/src/ee2x_update_suite/shared/manifest_builder.py`
- 现在规则分成两段：

1. 如果直接显式选择受保护文件/目录：
   - 继续报错
   - 文案：
     - `不能推送本机身份/登录状态文件；如需更新公共配置，请修改 Defaults/launcher-public.json。`

2. 如果选择整个启动器根目录：
   - 自动剔除 `Config / Logs / data/userdata / update/runtime`
   - 继续生成 launcher 包
   - 返回 warning：
     - `已自动排除启动器本地状态目录（Config/Logs/data/userdata/update/runtime），其余启动器文件将继续打包。`
