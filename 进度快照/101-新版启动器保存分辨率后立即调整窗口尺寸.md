# 进度快照 101 — 新版启动器保存分辨率后立即调整窗口尺寸

> 日期: 2026-05-26

---

## 已完成内容

### 1. 修复“保存分辨率后窗口尺寸不变化”的问题

- 之前 `分辨率` 只是被保存进配置文件
- 没有任何逻辑真正消费这个值来调整启动器窗口尺寸
- 现在保存设置后会立即按 `preferredResolution` 调整主窗口尺寸

### 2. 启动时也会按已保存分辨率恢复窗口尺寸

- `bootstrap_state` 读取配置时
- 会同步把主窗口尺寸调整到当前 `preferredResolution`
- 这样下次打开启动器时也能保持用户保存的窗口大小

### 3. 处理方式

- Rust 侧新增：
  - 分辨率字符串解析逻辑
  - 主窗口尺寸应用逻辑
- 当前支持直接解析形如：
  - `1024x768`
  - `1280x720`
  - `1366x768`
  - `1600x900`
  - `1920x1080`
- 调整尺寸后会尝试重新居中窗口

### 4. 已完成构建验证

- `npm run build` 通过
- Rust `release` 构建通过
