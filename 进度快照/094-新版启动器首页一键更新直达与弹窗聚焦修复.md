# 进度快照 094 — 新版启动器首页一键更新直达与弹窗聚焦修复

> 日期: 2026-05-26

---

## 已完成内容

### 1. 首页“一键更新”改为点击即直接执行强制更新

- 之前首页右侧的“一键更新”按钮只会：
  - 切到更新中心页面
  - 拉取更新信息
- 现在改为：
  - 点击后直接切到更新中心
  - 同时立即执行一次强制更新
- 不再需要用户进入更新页后再手动点第二次按钮

### 2. 弹窗遮罩已增加真实背景模糊与更重暗化层

- `overlay` 现在增加：
  - `backdrop-filter: blur(...)`
  - `-webkit-backdrop-filter: blur(...)`
  - 更深的背景遮罩色
- 目的：
  - 压暗背景内容
  - 弱化背景文字与日志干扰
  - 让弹窗操作区从背景中明确浮出来

### 3. 弹窗本体已加强聚焦效果

- `modal` 现在额外增加：
  - 更高不透明度的深色背景
  - 更强的外阴影
  - 更明确的层级感
- 这样更新完成弹窗、登录/注册弹窗、引导弹窗都会更清晰

### 4. 已重新正式编译并自动递增版本号

- 使用正式编译入口：
  - `npm run build:release`
- 版本号自动从 `1.0.3 -> 1.0.4`
- 已生成新安装包：
  - `EE2X魔改版启动器_1.0.4_x64-setup.exe`
