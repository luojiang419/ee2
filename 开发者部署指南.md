# 新开发者 Git 部署指南

> 将此文档提供给 AI 智能体（Claude Code 等），即可自动完成部署。

---

## 一、前置信息

请新开发者确认以下信息后，直接告诉智能体：

| 项目 | 说明 | 示例 |
|------|------|------|
| GitHub 用户名 | 协作者的 GitHub 账号 | `zhangsan` |
| 邮箱 | 用于 commit 记录 | `zhangsan@gmail.com` |
| 仓库地址 | HTTPS 格式 | `https://github.com/luojiang419/ee2.git` |
| 仓库权限 | 仓库所有者需先邀请协作者 | Settings → Collaborators → Add people |

---

## 二、智能体执行指令

> 将以下内容直接发送给 AI 智能体，智能体会按顺序执行每一步。

```
请按以下步骤配置本机的 Git 开发环境：

1. 检查 Git 是否安装（git --version），未安装则提示我

2. 配置用户信息：
   git config user.name "我的GitHub用户名"
   git config user.email "我的邮箱"

3. 添加远程仓库：
   git remote add origin https://github.com/luojiang419/ee2.git
   （如果已存在则跳过）

4. 拉取最新代码并覆盖本地：
   git fetch origin
   git checkout main
   git reset --hard origin/main

5. 设置分支追踪：
   git branch --set-upstream-to=origin/main main

6. 验证：git remote -v 和 git status 确认一切正常

7. 完成后提示我：在 VS Code Source Control 面板中，以后修改代码后写提交说明，然后点击 Sync Changes 即可一键推送。
```

---

## 三、仓库权限（仓库所有者操作）

在 GitHub 上为协作者添加写入权限：

1. 打开 https://github.com/luojiang419/ee2
2. **Settings** → **Collaborators** → **Add people**
3. 输入协作者的 GitHub 用户名
4. 协作者会收到邀请邮件，接受后即可推送代码
