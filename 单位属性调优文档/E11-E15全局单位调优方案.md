### 0.2 核心问题修正

**此前方案的核心缺陷**：14时代方案中各文明在某些维度存在"断崖式领先"——美国海军系数1.15、俄罗斯陆军系数1.20，导致特定文明在特定兵种上完全碾压，玩家缺乏反制手段。

**本轮修正原则**：
- **任何文明在任何维度不得超过基准值的 112%**（即最多领先12%）
- **弱势维度的劣势不超过基准值的 88%**（即最多落后12%）
- **差异体现在"性格"而非"绝对强度"**——HP高的速度慢、伤害高的装填慢、便宜的个体弱
- **通过RPS克制实现四两拨千斤**——正确克制的单位即使面板差12%，也能打出200%-400%伤害

## 一、中美俄现实军事投影

### 1.1 美国 — "技术制胜 · 精兵路线"

**现实军事特征**：
- **海军绝对优势**：11艘核动力航母、70+艘宙斯盾驱逐舰/巡洋舰、50+艘核潜艇。全球部署、远洋作战能力无可匹敌
- **空军代差优势**：F-22(187架)/F-35(600+架)五代机群，B-2/B-21隐身轰炸机，全球唯一真正隐身空军
- **陆军中等规模**：M1A2 Abrams约2500辆，强调信息化、联合兵种、精确打击。地面力量依赖海空掩护
- **技术投入巨大**：军费>$800B，单件装备造价高昂但性能顶尖
- **短板**：陆军数量不足以单独发动大规模地面战，野战防空依赖空军（"爱国者"系统不如俄S-400覆盖广）

**→ 游戏投影**：

| 维度 | 系数 | 性格描述 |
|:-----|:-----|:---------|
| 海军 | **1.10** | 质量最优、造价最贵、射程远、装填快 |
| 空军 | **1.10** | 隐身/速度/先发优势，造价最高 |
| 陆军 | **0.92** | 装甲偏薄、装填最快、灵活性最好、依赖协同 |
| 野战防空 | **0.90** | 对空伤害略低、依赖空军制空 |
| 防御建筑 | **0.92** | 海外基地模式，静态防御非优先 |
| 造价系数 | **1.12** | 精兵贵兵，单位数量少但个体强 |

### 1.2 俄罗斯 — "钢铁洪流 · 火力至上"

**现实军事特征**：
- **陆军传统强国**：T-14/T-90M/T-72B3约3000辆现役+10000辆储备，继承苏联"大纵深"战役理论
- **防空体系全球最完善**：S-400/S-500多层防空反导网，从战术到战略高度覆盖
- **火炮数量压倒性**：自行火炮/火箭炮数量全球第一，强调"火力摧毁而非精确打击"
- **海军严重萎缩**：仅1艘航母(库兹涅佐夫号，常年维修)、水面舰艇老旧、主力转向核潜艇
- **空军规模有限**：Su-57仅10+架量产，主力为Su-35/Su-30SM四代半
- **短板**：海军水面力量极弱、精确制导能力不足、电子战/信息化落后

**→ 游戏投影**：

| 维度 | 系数 | 性格描述 |
|:-----|:-----|:---------|
| 陆军 | **1.10** | HP最高、单发伤害最高、装填最慢、速度偏慢 |
| 野战防空 | **1.10** | 对空伤害最高、射程最远 |
| 防御建筑 | **1.08** | 国土防御传统，要塞/防空阵地耐久高 |
| 海军 | **0.88** | 水面舰艇HP/伤害偏低、速度慢、造价便宜 |
| 空军 | **0.90** | 数量弥补质量、HP偏低、伤害中等 |
| 造价系数 | **0.90** | 数量优势，单价便宜可大规模生产 |

### 1.3 中国 — "均衡崛起 · 反介入"

**现实军事特征**：
- **陆军规模最大**：约5000辆主战坦克(99A/96B)，世界最大常备陆军，快速现代化
- **海军急速扩张**：3艘航母(Liaoning/Shandong/Fujian)、Type 055全球最强驱逐舰之一、054A护卫舰量多质优
- **空军快速追赶**：J-20隐身战斗机(200+架)、J-16多用途、体系化AEW&C支援
- **火箭军独特优势**：DF-21D/DF-26反舰弹道导弹体系，区域拒止/反介入核心
- **短板**：航母舰载机经验不足、远程投送能力有限、部分装备代差仍在缩小

**→ 游戏投影**：

| 维度 | 系数 | 性格描述 |
|:-----|:-----|:---------|
| 陆军 | **1.05** | 强调机动力和射程，快打快撤 |
| 海军 | **1.00** | 均衡无短板，中规中矩 |
| 空军 | **0.98** | 快速追赶中，J-20隐身优势初现 |
| 野战防空 | **1.05** | HQ-9/HQ-17体系完善 |
| 防御建筑 | **1.05** | 要塞化防御思维 |
| 造价系数 | **1.00** | 居中，不便宜也不贵 |

## 二、时代升级幅度标准

### 2.1 核心规则

每升一个时代（E11→E12→E13→E14→E15），单位核心属性提升 **15%-25%**。具体幅度因单位类型而异：

| 单位类型 | 每代HP增幅 | 每代伤害增幅 | 说明 |
|:---------|:---------|:-----------|:-----|
| 主战坦克 | +18-22% | +18-22% | 陆军核心，稳步提升 |
| 轻型坦克 | +15-20% | +15-20% | 辅助定位，增幅稍低 |
| 装甲车/IFV | +15-18% | +12-18% | 轻甲快速，不强调HP |
| 火炮/坦歼 | +16-20% | +20-25% | 火力优先 |
| 水面主力舰 | +20-25% | +20-25% | 大型舰艇代差明显 |
| 小型舰艇 | +15-20% | +15-20% | 护卫舰/驱逐舰 |
| 潜艇 | +18-22% | +18-22% | 水下力量 |
| 航母 | +22-28% | +15-20% | 平台生存优先 |
| 战斗机 | +15-20% | +18-25% | 技术代差突出 |
| 轰炸机 | +15-20% | +20-25% | 载弹量提升 |
| 直升机 | +15-18% | +15-20% | 辅助平台 |
| 防御建筑 | +18-25% | +18-22% | 静态防御需跟随时代 |

### 2.2 造价增幅

每代造价提升幅度约为属性增幅的 **60%-80%**（性价比随时代提升，后期单位更划算）：

| 资源类型 | 每代增幅 |
|:---------|:---------|
| 基础资源(粮/木/石) | +8-15% |
| 黄金 | +10-18% |
| 特殊资源(石油/铀) | +12-20% |

### 2.3 E11→E15 全时代成长曲线示例（重型坦克线）

```
        HP      伤害     射程    装填    速度
E11:   1900     140      12     3.0     1.4
E12:   2500     195      13     3.0     1.3    (+31%/+39%) ← WWII重坦爆发
E13:   3400     260      14     3.0     1.2    (+36%/+33%)

## 三、RPS 强克制体系设计

### 3.1 设计目标

- **强克制关系**：克制方对被克制方造成 **200%-400%** 伤害
- **中立关系**：同类型或无关类型保持 **100%**（标准伤害）
- **被克制关系**：被克制方对克制方造成 **30%-75%** 伤害
- **防空/反舰特殊克制**：专用平台对特定目标有极端克制力

### 3.2 完整克制矩阵

#### 3.2.1 陆军内部克制

```
装甲车(LightMounted) ──克制──▶ 步兵(LightInfantry/HeavyInfantry)   [1.8x]
轻型坦克(HeavyMounted) ──克制──▶ 装甲车(LightMounted)              [2.0x]
主战坦克(HeavyMounted) ──克制──▶ 轻型坦克                          [1.6x]
重型坦克(HeavyMounted) ──克制──▶ 主战坦克                          [1.5x]
坦歼(LightArtillery) ──克制──▶ 重型坦克/主战坦克                   [2.5x]
步兵(HeavyInfantry2) ──克制──▶ 坦歼(薄甲开放式)                    [2.0x]
步兵(HeavyInfantry2) ──克制──▶ 建筑/要塞                           [1.8x]

## 十一、关键注意事项

1. **DDF LF换行符铁律**：所有DDF修改必须保持LF换行，使用PowerShell精确行号替换法
2. **CSV≠唯一数据源**：人口(popCount)、RPS类型、移动速度在DDF中定义，CSV修改可能不生效
3. **两个升级表并存**：`upgrade_unittypes.csv`(主)和`upgrade_unittypes_EE2X.csv`(资料片)需同步修改
4. **EXE硬编码限制**：某些文明属性白名单硬编码在EE2X.exe中，新增文明专属单位需验证
5. **Simulation目录**：RPS克制修改需确保工作数据库中有完整的Simulation目录
6. **资源时代合法性**：Oil(E10-E15)、Uranium(E13-E15)，造价中使用的特殊资源必须符合时代范围
7. **升级引用一致性**：dbtechtreenode.csv的UPGRADE值必须在upgrade_unittypes.csv中存在，否则静默失效
8. **AI自适应**：修改单位属性后，AI的.aip文件可能需要同步调整以确保AI能正确使用差异化单位
