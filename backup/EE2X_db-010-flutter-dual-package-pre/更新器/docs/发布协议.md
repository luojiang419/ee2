# EE2X 静态更新协议

## 目录结构

```text
{remoteRoot}/
└── updates/
    └── stable/
        ├── latest.json
        └── releases/
            └── {releaseId}/
                ├── EE2X-{releaseId}.zip
                ├── release-manifest.json
                └── release-notes.txt
```

## latest.json

字段：

- `schemaVersion`
- `channel`
- `version`
- `releaseNotes`
- `manifestUrl`
- `packageUrl`
- `packageSha256`
- `packageSize`
- `publishedAt`

## release-manifest.json

字段：

- `schemaVersion`
- `version`
- `rootDirName`
- `packageFileName`
- `packageSha256`
- `applyMode`
- `protectedPaths`
- `deleteList`
- `files`

`files` 每项包含：

- `path`
- `size`
- `sha256`

## 应用规则

1. ZIP 顶层目录必须为 `Empire Earth II/`
2. 只允许覆盖 manifest 中声明的文件
3. 只允许删除 `deleteList` 中声明的路径
4. `protectedPaths` 永远跳过，不允许被发布包覆盖
